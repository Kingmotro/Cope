# Cope
Cope is a simple and easy config format, parser and API for Java.

The config format is very simple but readable. It supports comments.

# Config format

```yaml
# Comment support
global:
    max-con 200

timeout:
	connect 5000
    client 50000
    server 50000

server:
    bind 0.0.0.0 80
```
